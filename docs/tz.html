Проєкт: Blog Platform API

1. Вступ

1.1 Мета документа

Даний документ є технічним завданням на розробку програмного продукту Blog Platform API. Метою документа є визначення функціональних та нефункціональних вимог до серверного API, що забезпечує роботу блог-платформи.

1.2 Область застосування

  • створення блог-платформи;
  • керування публікаціями;
  • взаємодія користувачів із контентом;
  • інтеграція з веб- та мобільними застосунками.

1.3 Терміни та скорочення

  • API — інтерфейс програмування додатків
  • REST — архітектурний стиль взаємодії
  • JSON — формат обміну даними
  • CRUD — Create, Read, Update, Delete
  • HTTP — протокол передачі даних

2. Загальний опис системи

Blog Platform API — це серверний додаток, що надає REST API для роботи з блог-платформою.

2.1 Основні функції системи

  • реєстрація користувачів;
  • авторизація;
  • створення, редагування, видалення та перегляд постів;
  • робота з коментарями;
  • пошук постів.

3. Функціональні вимоги

3.1 Реєстрація користувача

Система повинна приймати email і пароль, перевіряти коректність даних, створювати нового користувача та зберігати його у базі даних.

3.2 Авторизація

Система повинна перевіряти облікові дані, повертати токен доступу та забезпечувати захищений доступ.

3.3 Управління постами

  • створення поста;
  • отримання списку постів;
  • отримання одного поста;
  • редагування поста;
  • видалення поста.

3.4 Робота з коментарями

Система повинна дозволяти додавати, переглядати та видаляти коментарі.

3.5 Пошук

Система повинна виконувати пошук за назвою поста та фільтрацію за автором.

4. API endpoints

  • POST /register
  • POST /login
  • GET /posts
  • POST /posts
  • PUT /posts/{id}
  • DELETE /posts/{id}
  • GET /comments
  • POST /comments

5. Нефункціональні вимоги

  • обробка запитів до 2 секунд;
  • стабільність роботи;
  • захист даних користувача;
  • масштабованість системи;
  • сумісність з web та mobile клієнтами.

6. Тестування

  • перевірка реєстрації;
  • перевірка авторизації;
  • перевірка CRUD для постів;
  • перевірка коментарів;
  • перевірка помилок.
